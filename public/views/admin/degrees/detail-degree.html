Computer Science program for students starting in Fall 2015 with CS 115

This degree requires the student to take all of the CS Core classes.
  1. Introduction to Computer Science (CS115)
  2. Data Structures (CS284)
  3. Systems Programming (CS392)
  4. Software Development Process (CS347)
  5. Concurrent Programming (CS511)
  6. Operating Systems (CS492)
  7. Automata and Computation (CS334)
This degree requires the student to take at least 1 of the Software Development Elective classes.
  1. Web Programming (CS546)
  2. TCP/IP Networking (CS521)
  3. Interactive Computer Graphics (CS537)
  4. Distributed and Cloud Computing (CS549)
  5. Compiler Design (CS516)
  6. Computer Vision (CS558)
This degree requires the student to take the following 300 Level Humanity Elective classes.
  1. Science and the Media (HST320)
  2. Modern Literature (HLI312)
  3. Science Fiction (HLI316)
